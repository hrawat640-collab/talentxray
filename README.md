# talentxray

## Run locally (frontend + backend)

1. Install dependencies:
   - `npm install`
2. Start server:
   - `npm run dev`
3. Open:
   - [http://localhost:3000](http://localhost:3000)

## Backend endpoints

- `GET /api/health`
- `POST /api/lead`
- `POST /api/login-event`
- `POST /api/search-event`

Events are stored locally in `data/*.jsonl`.
